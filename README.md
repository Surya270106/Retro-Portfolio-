# 🎮 Retro-Portfolio

A highly interactive, cinematic personal portfolio built with Vanilla HTML, Tailwind CSS, and GSAP. 

This portfolio blends modern web design aesthetics with deep nostalgic elements. It features a fully playable 8-bit Snake game built into a floating, Minecraft-inspired Gameboy, complete with a massive cinematic scroll-zoom transition that literally pulls the user inside the device!

## ✨ Features
* **Playable Gameboy Console**: A custom-built, interactive Gameboy with a fully working Snake game (HTML5 Canvas). Features functional D-Pad/A/B buttons, wrapping screen logic, and high score local-storage tracking!
* **Cinematic Scroll Transition**: Powered by GSAP ScrollTrigger, scrolling down the page locks the viewport, massively scales the Gameboy 40x, and seamlessly transitions the user *inside* the Gameboy screen into the dark-themed project sections.
* **Liquid Glassmorphism**: Custom SVG displacement filters and backdrop blurs to create "liquid glass" brutalist UI elements.
* **Dynamic Breathing Gradients**: A custom vanilla JavaScript translation of Framer Motion's animated radial gradients, providing a subtle, breathing glow strictly around the borders of the footer.
* **Zero-Framework Architecture**: Built entirely without heavy frameworks—just pure, heavily optimized Vanilla JavaScript, raw CSS power, and GSAP.

## 💡 Inspirations & Aesthetics
This project was born out of a desire to merge several contrasting design languages into one cohesive, WOW-factor experience:
1. **Retro Handheld Gaming**: The green screen color palette (`#a6ca11`, `#0f380f`) and dot-matrix menu aesthetics pay direct homage to the original Nintendo Gameboy.
2. **Minecraft & Voxel Art**: The structural casing of the Gameboy features hard-edged, blocky shadows and a blue palette heavily inspired by Minecraft's iconic block aesthetics.
3. **Apple-style Cinematic Scrolling**: The massive, perspective-shifting scroll-zoom transition was inspired by high-end tech landing pages, designed to make the transition between the Hero section and the Work section feel like a physical journey.
4. **Brutalist Glassmorphism**: Combining the harsh, thick borders of Neo-Brutalism with the sleek, glowing translucency of Glassmorphism.

## 🛠️ Tech Stack
* **Core**: HTML5, CSS3, Vanilla JavaScript
* **Styling**: Tailwind CSS (for rapid utility classes combined with heavily customized CSS variables)
* **Animation**: GSAP (GreenSock) & ScrollTrigger for timeline-based scroll animations
* **Fonts**: Google Fonts (`Press Start 2P` for the 8-bit feel, `Inter` for modern typography)

## 🚀 Quick Start
Because this is a vanilla HTML project, there is no build step required!
1. Clone the repository
2. Open `index.html` in your browser.
3. Scroll down and enjoy the ride!
